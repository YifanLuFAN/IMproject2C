This week's inspiration

I found a website called openprocessing! There are websites made by people from all over the world using processing, and some of them also provide source code for me to learn from. This is really useful for me!

We learned in class this week how to create circles, change background colors, etc. in Processing. Based on this, I found a very interesting website in OpenProcessing. On a black background, there are 25 circles of different colors evenly distributed (the colors will be different every time you open the website). When the user's mouse moves between two circles, they will be connected by another circle, and the color of the connected circle depends on the two connected circles. Users can connect up to four circles at the same time. This is a great idea and an extension of the exercises we have done in class.
