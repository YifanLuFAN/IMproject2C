This week's inspiration

This week we learned to use Processing to capture user input (especially mouse and sound), so I found two related websites. The first one is a simple loudness graph. When the user speaks, the three lines on the screen will vibrate with different amplitudes according to the volume of the sound.

See the website

The second one is very interesting. Users can use the mouse to generate a spectrum of different colors and sizes. The main thing that inspired me about this website is its excellent feedback to the user's mouse input. When the user tries to click, slide, or keep the mouse, the website will react differently. This kind of excellent design in details can greatly improve the sense of interaction between the user and the website.
