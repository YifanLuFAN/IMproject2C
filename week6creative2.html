User input, I think this is the place that best reflects the interactive design of a website. We learned about mouse input and sound input this week.
Regarding the sound input, I got inspiration from openprocessing. When the user's voice becomes louder, the images that were originally overlapped will separate, creating a sense of movement. This is very suitable as feedback for the user's sound input. The image processing is done by the mask function in Photoshop.
