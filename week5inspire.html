This week's inspiration

This week we need to draw a Kusama sketch, and my work is based on Kusama's "Phosphoresce in the Daytime". This work retains Kusama's usual painting habits - filling the picture with dots, but the difference is that we can see three larger circular blocks of color (blue, yellow, and red) in the middle of the picture. This idea inspired me. I also learned to use circular blocks of color to draw like Kusama, but I have more blocks of color and the colors I choose are closer.

This week I also found an interesting website! There is nothing on the webpage at first. When the user clicks somewhere, a "point" is created. This point will emit curves in all directions. When the emitted curve reaches a certain length, a new "point" will be generated, and the newly generated points will repeat the process. I think this is simulating cell division. This idea of "click to create - creation continues to create" has inspired me very well.
