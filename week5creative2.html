Creative coding3

This week we also learned to capture the user with the camera, and I love this feature.
Based on the code built in class (turning the user image black and white), I incorporated the inspiration from Kusama's work into creative coding - turning the image into a dot image. This dot image has a very good visual effect and also has a certain "blurring" effect.
In addition, I also added the function of changing the size of the dots by moving the mouse. In this way, users can choose the image they want between "clear" and "overexposed"!
